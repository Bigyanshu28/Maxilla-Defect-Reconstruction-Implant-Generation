# Maxilla Defect Reconstruction & Implant Generation

An end-to-end deep learning pipeline and web interface for automating patient-specific implant generation for maxillary defects. This tool ingests `.nrrd` files of defective maxillas, processes them through a 3D U-Net architecture, and outputs a 3D-printable `.stl` file of the required implant.

🔗 **Live Interface:** [[Link to Interface]](https://easiofy-detections.netlify.app)

## 🚀 Features
* **Automated Implant Generation:** Predicts the exact geometry of missing maxillary bone structure.
* **Clinical-Grade Processing:** Handles artifact and noise removal from segmented maxilla scans.
* **Direct-to-Print Output:** Converts predicted voxel masks into 3D-printable `.stl` meshes.
* **User-Friendly Interface:** Streamlined web workflow for uploading scans and downloading implants.

## 🧠 Technical Pipeline

### 1. Data Preparation & Preprocessing
* **CT to NRRD:** Raw CT DICOM files are segmented to isolate the maxilla and exported to the `.nrrd` format.
* **Denoising:** Preprocessing algorithms are applied to remove scan artifacts, noise, and non-bony structures.
* **Training Data Generation:** To train the model robustly, synthetic defects were procedurally generated by injecting clinically validated defect masks into intact, healthy maxilla scans.

### 2. Deep Learning Model 
* The core prediction engine is a model designed for volumetric segmentation. 
* It takes the preprocessed, defective 3D `.nrrd` volume as input and predicts a binary voxel mask representing the missing bone (the implant).

### 3. Post-Processing & Mesh Generation
* **Voxel-to-Mesh:** The output voxel mask from the 3D U-Net is passed through a surface reconstruction algorithm (e.g., Marching Cubes) to generate a smooth 3D mesh.
* **STL Export:** The resulting mesh is saved as a `.stl` file, ready for 3D printing and clinical evaluation.

## 💻 Interface Usage

1. **Navigate to the Interface:** Go to the [Home Page](#).
2. **Select Module:** Choose the **Maxilla Reconstruction** workflow.
3. **Upload Scan:** Upload the `.nrrd` file of the defected maxilla.
4. **Initiate Generation:** Select the **Implant Generation Module** and click **Proceed**.
5. **Predict:** Click **Predict** to run the scan through the backend 3D U-Net.
6. **Download:** Once processing is complete, download the generated implant in `.stl` format.
